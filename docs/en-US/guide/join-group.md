# Join Group

If you have questions, it is recommended to use `Github` for feedback. If `Github` and other platforms cannot solve your problem, you can join the group. Please ensure your questions are clear and polite. Thank you very much for your support!


## Communication Cases

Here are a few communication cases. Through these cases, we will understand which questioning methods are most likely to solve problems.

### Wrong Cases

**_Casual speech when asking questions, not reading documentation, not providing reproduction demos, not providing detailed problem descriptions, full of doubt and complaints, such as:_**

- Wrong case 1: This button is really逆天 (against heaven), `custom-class` doesn't work?
- Wrong case 2: Why is your component so garbage, the `custom-class` attribute doesn't work, I can't customize styles, can you fix it?
- Wrong case 3: ? This component has a bug? My custom styles don't work???


### Correct Cases

**_Carefully check the documentation and FAQ to confirm it is not resolved before asking. Describe your problem in detail when asking, and provide a reproduction demo for others to help troubleshoot. For example:_**

I encountered a problem where custom styles don't take effect when using the `custom-class` attribute of the `Button` component. I read the official documentation and FAQ, but still can't solve it. I've organized my minimum reproduction demo into a reproduction repository. Can you help troubleshoot the problem?

***Quick Tips for Locating Problems***
1. Read component documentation to see if anything is missed.
2. Check [FAQ](/guide/common-problems) for similar problems.
3. Check [Github issues](https://github.com/wot-ui/wot-ui/issues) for similar problems.
4. Check [Changelog](/guide/changelog) for related updates.
5. Learn [How to Ask Questions The Smart Way](https://lug.ustc.edu.cn/wiki/doc/smart-questions/) to quickly ask the right questions.
6. The exchange group mainly provides a platform for mutual aid and exchange. Please communicate friendly, don't be a freeloader, thank you for your support.
