# Introduction

In `V1` version, `Wot UI` is built based on `vue3`+`Typescript`, following the design specifications of `wot design`.

In `V2` version, we pay more attention to aesthetics and AI-friendliness, adopting a design with higher AI-friendliness. We provide 80+ high-quality components and AI programming practice solutions, supporting dark mode, internationalization, and custom themes, aiming to provide developers with concise and efficient UI interactions and an `AI-friendly` development experience.

## ✨ Features

- 🎯 Multi-platform coverage, supports WeChat Mini Program, Alipay Mini Program, DingTalk Mini Program, H5, APP, etc.
- 🚀 80+ high-quality components, covering mainstream mobile scenarios.
- 💪 Built with Typescript, providing a good component type system.
- 🤖 Provides AI-friendly design system.
- 🌍 Supports internationalization, built-in 15 language packs.
- 📖 Provides rich documentation and component examples.
- 🎨 Supports theme customization by modifying CSS variables.
- 🍭 Supports dark mode
